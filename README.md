# RNA-seq-pipeline
This is a RNA seq pipeline developed in snakemake work engine. All the file names have are general and no data is added because of privacy of our collaborators data. The organims is Mycobacterium Tuberculosis Strain H37Rv and it is advised to change the names and folders whereever required.
# Additional requirements:
Perform initial quality control check for the reads.
Install snakemake, fastp , bowtie2 , samtools , featureCounts and R
# Command
snakemake --cores 8
